# QTE TEST

## Run these commands when cloning this project

Install all node dependencies for server side, from root use:

### `cd server`

### `npm install`


... and to start server use:
### `npm start`

  
Install all node dependencies for client side, from root use:

### `cd client`

### `npm install`


... and to start server use:
### `npm start`


... and smile :)
